Android 源码编译

发表于 2016-03-22   |   分类于 Android   |     |   阅读次数

下载源码

根据清华AOSP镜像给出的文档在下载机的 VB 中的 Ubuntn 下载好源码
源码下载完毕有37G
压缩代码tar zcvf Source.tar.tgz WORKING_DIRECTOY
然拷贝到rmbp

配置

硬件 rmbp 840
系统 OS X Yosemite 10.10.5
JDK 1.7.0_15
Command Line Tools Xcode 6.4(6E35b)

下载binaries

Binaries for Nexus Devices, 选择自己机器和分支对应的 driver, 我这里对应的是Nexus 6 (Mobile) binaries for Android 6.0.1 (MMB29S)，3个都下载解压到 WORKING_DIRECTOY 目录后，分别运行三个 extract 开头的sh脚本.

开始编译

lunch aosp_shamu-userdebug 是选择需要编译的版本，直接输入 lunch 可以列出所有的可选项，选择需要的版本即可
