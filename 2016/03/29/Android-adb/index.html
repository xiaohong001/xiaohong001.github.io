一些 adb 命令

发表于 2016-03-29   |   分类于 Android   |  

adb 命令

  1. 列出现在运行的所有进程
    adb shell ps -x
  2. 查看内存使用
    adb shell dumpsys meminfo
  3. 挂载读写System
    adb shell mount -o remount,rw /system
  4. 新建文件夹
    adb shell mkdir /sdcard/test
  5. 列出所有的连接设备
    adb devices
  6. 无线连接设备
    adb connect <host>[:<port>]
  7. 拷贝文件/目录到设备
    adb push <local> <remote>
    拷贝文件夹的时候后面不能带\
  8. 从设备拷贝文件/目录
    adb pull <remote> [<local>]
  9. 只有发生改变时从主机拷贝到设备
    adb sync [<directory>]
  10. 运行远端shell交互
    adb shell
  11. 运行远端shell 命令
    adb shell <command>
  12. 运行仿真控制台命令
    adb emu <command>
  13. 浏览设备日志
    adb logcat [<filter-spec>]
  14. 转发套接字连接
    adb forward <local> <remote>
  15. 拷贝文件包到设备并安装
    adb install [-l] [-r] [-s] <file>
  16. 卸载程序包
    adb uninstall [-k] <package>
    -k意味着保留数据和缓存
  17. 返回所有的bugreport信息
    adb bugreport
  18. 查看帮助
    adb help
  19. 查看版本
    adb version
  20. 阻塞直到设备上线
    adb wait-for-device
  21. 启动adb
    adb start-server
  22. 关闭adb
    adb kill-server
  23. 列印offline|bootloader|device信息
    adb get-state
  24. 获取设备序列号
    adb get-serialno
  25. 连续列印设备状态
    adb status-window
  26. 重装载/system分区
    adb remount
  27. 重启
    adb reboot [bootloader|recomry]
  28. 重启到bootloader
    adb reboot-bootloader
  29. 切换到root
    adb root
  30. 切换到usb
    adb usb
  31. 停止应用
    adb shell am force-stop <package>
